# The Basics

## Creating and organizing

- Press **⌘N** to create a new note.
- Use folders in the sidebar to organize. Right-click for new note, new
  folder, rename, and delete.
- Rename inline by double-clicking a name in the sidebar.

## Writing

- Just start typing. Markdown formats in place as you write.
- Press **/** for the block menu: headings, lists, tables, code blocks,
  quotes, and more.
- Select text to get the formatting toolbar for bold, italic, links, and so
  on.
- Drag the handle on the left of a block to reorder it.

## Saving

- Autosave writes your changes about a second after you stop typing.
- Press **⌘S** to save immediately.
- Opening a note never changes it.

## Keyboard shortcuts

| Shortcut | Action |
| --- | --- |
| ⌘K | Command palette and search |
| ⌘N | New note |
| ⌘S | Save now |
| ⌘\ | Toggle sidebar |

## Themes

Switch between light and dark using the toggle in the top bar.

# Power Features

## Command palette

Press **⌘K** to jump to any note by name or run a command. It is the fastest
way to move around.

## Full-text search

Search across every note in the current workspace from the sidebar.

## Wiki links and linked mentions

- Type **[[** to link to another note.
- Each note lists its "Linked mentions" underneath, so you can see what points
  back to it.
- Links are saved as standard relative markdown links, so they stay portable.

## Properties and tags

- YAML frontmatter at the top of a note (title, tags, dates) appears as an
  editable properties table.
- Tags show as chips you can click.
- This very note has properties. Look at the top of the page.

## Comfort

- Resize the sidebar by dragging its edge.
- Light and dark themes.

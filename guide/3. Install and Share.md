# Install and Share

Use this page to install DXEditor, or to share the steps with a teammate.

## Download

Get the latest version from the releases page:

https://github.com/dxnmus/dxeditor/releases/latest

- **Mac (Apple Silicon, M1 or newer):** download the file ending in
  `arm64.dmg`.
- **Mac (Intel):** download the file ending in `x64.dmg`.
- **Windows:** download the file ending in `.exe`.

## Install on Mac

1. Open the downloaded `.dmg`.
2. Drag the DXEditor icon onto the Applications folder.
3. Eject the disk image.

The app is not from an identified developer yet, so the first open needs one
extra step:

1. Open Applications and find DXEditor.
2. Right-click it and choose **Open**, then click **Open** in the dialog.
3. If macOS blocks it instead, open **System Settings > Privacy & Security**,
   scroll down, and click **Open Anyway** next to DXEditor.

After that first time, it opens normally with a double-click.

If macOS ever says the app is "damaged," that is Gatekeeper being strict about
an app it cannot verify, not real damage. Clear it once from Terminal, then
open the app normally:

    xattr -dr com.apple.quarantine /Applications/DXEditor.app

## Install on Windows

1. Run the downloaded `.exe`.
2. If Windows shows a "Windows protected your PC" screen, click **More info**,
   then **Run anyway**. This appears only because the app is not code-signed
   yet.
3. The installer finishes and opens the app on its own.

## What happens on first launch

You land on this guide, plus a "My Notes" workspace to write in. To connect a
shared folder, use the workspace dropdown at the top of the sidebar and choose
**Add workspace**. Your files stay as plain markdown on your disk. Nothing is
uploaded anywhere.

See [Updating the App](<4. Updating the App.md>) for how new versions arrive.

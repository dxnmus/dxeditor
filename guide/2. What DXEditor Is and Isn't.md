# What DXEditor Is and Isn't

## What it is

- A local writing app for a folder of plain markdown files.
- Yours. Files stay as `.md` on your disk. Nothing is uploaded to any server.
- Team-friendly through OneDrive sync, with a safety net so you do not
  overwrite each other.
- A Notion-style editor: type and it formats in place, with a block menu and
  a formatting toolbar.

## What it is not

- Not a cloud app or a website. There is no account and no server you log
  into.
- Not real-time co-editing like Google Docs. The model is "we each work on
  different notes," and OneDrive syncs them. If two people touch the same
  note, the app stops the second save from erasing the first.
- Not a general file manager. It only shows and edits `.md`, `.markdown`,
  `.mdown`, and `.txt` files.

## Why it works this way

Your notes outlive the app. Because everything is plain markdown on disk, you
can open the same files in any other editor, back them up, or move them, and
nothing is locked inside a proprietary format.

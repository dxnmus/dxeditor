# What's New

The newest changes are at the top. Each entry matches an app version.

## v0.1.5 — July 2026

- Right-sized the heading fonts so documents read with a cleaner, more
  balanced hierarchy.

## v0.1.4 — July 2026

- Refreshed, more modern look: a cleaner window with the app logo at the top
  of the sidebar, softer labels, and rounded selection highlights.
- Formatting and commenting now share one compact toolbar that appears when
  you right-click selected text, so nothing pops up while you read.
- Comment highlights are easier to see, and open folders are tinted so you can
  scan the tree at a glance.

## v0.1.3 — July 2026

- The DXEditor Guide now stays pinned at the bottom of the sidebar at all
  times, so it is always one click away.
- Fixed automatic updates: releases now publish completely, so the app can
  find and install new versions.

## v0.1.2 — July 2026

- Fixed the "app is damaged" message some Macs showed on first launch. The
  Mac app now carries a valid signature, so it opens with the normal
  first-run prompt instead.

## v0.1.1 — July 2026

- Automatic updates are live: the app now checks for new versions and lets you
  install them without hunting for a download.
- Clearer messaging when an update check cannot reach the server.

## v0.1 — July 2026

- First desktop release for macOS and Windows.
- Built-in guide (these pages), which refreshes automatically when you update
  the app.
- Automatic update checks against GitHub Releases.
- Everything from the web app: workspaces, Notion-style editing, comments for
  review, wiki links, properties, full-text search, team-safe saving, and
  light and dark themes.
